Coat

Coat is a tiny css file, which you can include to make pages look nice.

Like this page, it's build up using completely semantic html without any classes or styling (besides coat) and still looks pretty good!

Installation

You can either add it in your html > head as 

<link rel="stylesheet" type="text/css" href="https://coat.guyn.nl/style.css" />

or import it directly in your stylesheet; 

@import "https://coat.guyn.nl/style.css";

Settings

There are a few you things you can edit, like the colors and borders.

Variable Default value
--color-primary rgba(50, 180, 200, 0.25)
--color-secondary rgba(180, 70, 145, 0.25)
--border-color rgba(200,200,200,1)
--border-radius 0.25em
--code-background var(--color-primary)
--code-color currentColor
--body-max-width 960px
